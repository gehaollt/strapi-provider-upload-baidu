# Strapi-Provider-Upload-Baidu

A baidu bos provider plugin deved for strapi.

[BOS Node.js SDK](https://cloud.baidu.com/doc/BOS/s/Djwvyrhiw)

**Example**

```json
./extensions/upload/config/settings.json
{
  "provider": "baidu",
  "providerOptions": {
    "ak": "AccessKey",
    "sk": "SecretAccessKey",
    "domain": "Domain URL",
    "bucket": "my-bucket"
  }
}
```
